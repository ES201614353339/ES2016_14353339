### 具体修改的方法

example1
```
    将 square.c 里面的 i = i*i；修改成 i = i*i*i；
	这样就能顺利的把三次方改成平方
```
example2
```
    将 example2.xml 中的 value="3"；修改成 value="2"；
	这样就能顺利的把三次平方改成两次平方
```
